# 内核设计 Wiki

本 Wiki 基于《如何设计内核：权限、调度与结构》一书的内容，按概念组织为独立页面，方便快速查询。

**三个核心模型：**

- [**权限模型**](permission/README.md) — 谁可以用？允许还是禁止？
- [**调度模型**](scheduling/README.md) — 资源什么时候给谁、用多久？
- [**结构模型**](structure/README.md) — 内核如何组织自己？

**编辑方式：** 本 Wiki 的所有内容都是 Markdown 文件，存放在 [GitHub 仓库](https://github.com/dslsdzc/kernel-design-book/tree/main/wiki/src) 中。如需修改或补充，直接提交 Pull Request。
