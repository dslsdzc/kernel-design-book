- [能力模型](capability.md) — 基于能力的授权
- [标签模型](label.md) — 基于标签的访问控制
- [信任锚](root-of-trust.md) — 权限的起点
- [权限的传递](delegation.md) — Move / Copy / Mint
- [权限的撤销](revocation.md) — 收回已授予的权限
- [权限的衰减](attenuation.md) — 只给一部分
