# 贡献指南

本 Wiki 的所有内容以 Markdown 文件形式存放在 [GitHub 仓库](https://github.com/dslsdzc/kernel-design-book/tree/main/wiki/src) 中。

## 编辑流程

1. Fork 仓库
2. 在 `wiki/src/` 目录下编辑或新建 `.md` 文件
3. 如果新建页面，需要在 `SUMMARY.md` 中添加条目
4. 提交 Pull Request

## 页面约定

- 每个页面聚焦一个概念
- 页面前几行简要定义该概念
- 使用 `> 对应书籍：第 X 章` 注明来源
- 代码块标注语言
